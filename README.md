# CP01 - JavaAdvanced
Este projeto é o frontend de uma aplicação de biblioteca virtual, desenvolvido com Angular.
A aplicação permite a visualização e interação com livros, usuários e funcionalidades relacionadas ao sistema.

# Como rodar o projeto
```shell
git clone https://github.com/JoaoVictor087/Biblioteca.git
cd Biblioteca
npm install
ng serve
```

Após isso abra o navegador e acesse 
```http://localhost:4200/```

Esse projeto utiliza uma API REST desenvolvida em Java.
Certifique-se de que a API esteja rodando antes de acessar o frontend.
link da API:
[link da API](https://github.com/vinikenzo/Biblioteca-api)
